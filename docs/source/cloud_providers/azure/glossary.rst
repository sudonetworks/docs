.. glossary::

   Virtual Machine (VM)
      An emulation of a physical computer, allowing you to run operating systems and applications.

   Blob Storage
      Microsoft Azure's object storage solution for storing large amounts of unstructured data.

   Virtual Network (VNet)
      A representation of your own network in the cloud. It is a logical isolation of the Azure Cloud dedicated to your subscription.
