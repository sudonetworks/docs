.. glossary::

   VLAN
      Virtual Local Area Network. A logical grouping of network devices that allows them to communicate as if they were on the same physical network segment.

   ACL
      Access Control List. A set of rules used to filter network traffic based on source and destination IP addresses, ports, and protocols.

   OSPF
      Open Shortest Path First. A routing protocol used to find the best path for data packets within an autonomous system.
